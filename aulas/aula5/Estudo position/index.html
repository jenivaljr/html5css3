Position

O valor position: static não se posiciona de forma especial

Position: relative se comporta igual ao static, exceto se for adicionado as propriedades (top, right, bottom e left no estilo do elemento).

Definindo propriedadese um elemento position: relative, ajustamos o elemento em relação a postição dele. Outros elementos não vão se ajustar automaticamente para se posicionarem em qualquer espaço deixado.
